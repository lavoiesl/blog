--- title: "Starting a text editor from inside a Virtual Machine" tags: ["Development tool", "Vagrant"] date: 2014-11-05 17:24:00 -0500 --- Using Vagrant, Docker or other virtual development environment is becoming quite popular. However, a drawback of this is that you cannot start a visual text editor on your main machine because heh, the files are not there. Purist may tell you that you should be using Vim or Emacs, but I like Sublime, I want to keep using it.
